== Job Tracker - Angular


This application uses Rails for backend REST calls with Angular on the frontend.
Utilising the Clearance gem for authentication and authorization.

App Specs:

* Ruby v2.2.2

* Rails v4.2.4

* Databae: Postgres

* bootstrap-sass (3.3.7)

* AngularJS v1.6.1
